# ⚡ Node.js & GraphQL CRUD API

A beginner-friendly GraphQL API built with Node.js, Express and MongoDB as part of a structured backend workshop series created in collaboration with the Scientific Club at my university.

---

## Overview
This project is part of a collaborative workshop series where I contributed as an instructor, helping design and deliver sessions focused on teaching students how to build backend APIs.

Each session of the workshop covered a different backend technology (MongoDB, GraphQL, Prisma, Sequelize, SQL) and this project represents the **Node.js + GraphQL module**.

The goal was to introduce students to real-world backend development using GraphQL APIs.

---

## Features

This API provides basic CRUD operations using GraphQL:

### 👤 User Management
- User registration
- User login
- Get all users
- Get user by ID

### 📝 Post Management
- Create a post
- Get posts
- Update post
- Delete post

### 💬 Comment Management
- Create comments
- Link comments to posts and users

---

## Authentication & Validation
- JSON Web Token (JWT) authentication
- Password hashing using bcrypt
- GraphQL schema validation and resolvers logic
