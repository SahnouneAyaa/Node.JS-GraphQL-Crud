# Node.js & GraphQL CRUD Tutorial

A beginner-friendly tutorial created in collaboration with the Scientific Club at my university to teach students how to build a GraphQL API with Node.js.

The project demonstrates CRUD operations, helping students understand:

- How to define GraphQL schemas and resolvers  
- Querying and mutating data efficiently  
- Integrating Node.js with MongoDB using Mongoose  
- Best practices for API design with GraphQL


## Tech Stack

- Node.js  
- Express.js  
- GraphQL
- MongoDB  
- Mongoose
